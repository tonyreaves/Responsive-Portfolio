My story

I spent a decade in journalism, working as a reporter, copy editor, web editor and social media manager. I'd long been interested in web development and, in late 2017, I began in earnest to learn coding.


I've already built several pages and apps. You can check them out on my portfolio page. I'm proficient in JavaScript, and projects in my portfolio use jQuery, JSON and Node.js. I've used CSS,Bootstrap and Materialize in my layouts.


As a journalist, I was dedicated to making information engaging and easy to understand. I'm bringing that same philosophy to my web development.

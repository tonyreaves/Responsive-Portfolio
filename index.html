My story

I spent a decade in journalism, working as a reporter, a copy editor, a web editor and a social media manager. In late 2017, I wanted a change and began learning web development.


I've already built several pages and apps. You can check them out on my portfolio page. I'm proficient in JavaScript, and projects in my portfolio use jQuery, APIs and Node.js. I've used CSS, Bootstrap and Materialize in my layouts.


As a journalist, I was dedicated to making information engaging and easy to understand. I'm bringing that same philosophy to my web development.
